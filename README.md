# STRING.THEORY.EXETER

Premium racket stringing & customization website with brutalist design and interactive animations.

## Project Structure

```
string_theory_exeter2/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Styles and animations
├── js/
│   └── main.js         # JavaScript functionality
├── images/             # Photos and images
└── README.md
```

## Overview

Single-page website featuring:
- Hero section with animated polaroid photos
- Services, promotions, and live Google Sheets stock integration
- Team profiles with images
- Image gallery
- Contact information

## Design

**Brutalist aesthetic** with lighter color palette, bold typography, thick borders, and interactive animations including floating polaroids, hover effects, and scroll-triggered fade-ins.

## Features

- ✅ Brutalist design
- ✅ Fully responsive
- ✅ Animated polaroid backgrounds
- ✅ Interactive hover effects
- ✅ Live Google Sheets integration
- ✅ Smooth scroll navigation
- ✅ Vanilla JavaScript (no frameworks)

## Technical Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Inter, Montserrat)
- CSS Grid & Flexbox
- IntersectionObserver API
